Elevation above creek

This tool will calculate the elevation of each grid cell in a digital elevation model (DEM) above the first creek cell reached by following the flowpath (McGuire et al., 2005; Seibert and McGlynn, 2005).

The user must specify:
The name of the input digital elevation model (DEM).
The name of the input streams raster.
The name of a Travel Time control (TT-control) input raster (optional).
The name of the Elevation Above Creek (EAC) output raster.
The name of the Distance From Creek (DFC) output raster.
The name of the Gradient To Creek (GTC=EAC/DFC) output raster.
The name of the Travel Time Proxy (TTP) output raster.

NoData valued grid cells in the input pointer image will be assigned NoData values in the output image. The output raster is of the float data type and continuous data scale.

References:

McGuire, K. J., J. J. McDonnell, M. Weiler, C. Kendall, B. L. McGlynn, J. M. Welker, and J. Seibert (2005), The role of topography on catchment-scale water residence time, Water Resour. Res., 41, W05002, doi:10.1029/2004WR003657

Seibert, J. and B. McGlynn, 2005. Landscape element contributions to storm runoff. In: Anderson, M. G. and J. McDonnell (Eds.). Encyclopaedia of Hydrologcial Sciences, John Wiley & Sons, Ltd, Chichester, 1751- 1761
