Maximum Downslope Elevation Change

This tool calculates the maximum elevation drop between each grid cell and its neighbouring cells. The user must specify the name of the input DEM and the output raster name. The user must specify the names of the input digital elevation model (DEM) and output raster. NoData valued grid cells in the input pointer image will be assigned NoData values in the output image. The output raster is of the float data type and continuous data scale.
