Fill depressions by size

This tool can be used to fill topographic depressions, up to a maximum user-specified size, in a digital elevation model (DEM). It also removes the associated flat areas. This is a common pre-processing step required by many flow-path analysis tools to ensure continuous flow from each grid cell to some outlet point. The depression filling algorithm is essentially the same as the algorithm implemented in the Fill Depressions tool. Unlike the Fill Depressions tool, however, this tool only ever applies the filling algorithm on a series of small, overlapping sub-sections of the input DEM, the size of which is determined by the maximum user-specified size. As such, this tool is generally quicker for filling depressions in datasets containing only small-sized depressions. It can also be used in combination with the Depression Breaching tool.

If the input DEM has gaps, or missing-data holes, that contain NoData values, it is better to use the Fill Missing Data Holes tool to repair these gaps. Fill Missing Data Holes toolwill interpolate values across the gaps and produce a more natural-looking surface than the flat areas that are produced by depression filling.
