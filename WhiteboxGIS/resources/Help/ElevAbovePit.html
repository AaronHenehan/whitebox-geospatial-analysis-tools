Elevation above pit

This tool will calcuate the elevation of each grid cell in a digital elevation model (DEM) above the nearest downslope pit cell or grid edge cell, depending on which is encountered first during the flow-path traverse. The resulting image is therefore a measure of relative landscape position. The user must specify the names of a D8 flow pointer grid, a DEM file, and the output file. The flow pointer grid must be derived using the D8 flow algorithm.

NoData valued grid cells in the input pointer image will be assigned NoData values in the output image. The output raster is of the float data type and continuous data scale.
