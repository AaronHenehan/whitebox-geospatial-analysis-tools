Set NoData value

This tool will re-assign a user-defined background value in an input raster image the NoDatavalue (-32,768). Most Whitebox GAT tools recognize NoDatagrid cells and treat them specially. NoDatagrid cells are also displayed differently. The user must input the name of the raster image to be updated and the current background value. The input image will be altered such that all grid cells containing the background value will be replaced with the NoDatavalue. The default background value is zero, although any numeric value is possible.
